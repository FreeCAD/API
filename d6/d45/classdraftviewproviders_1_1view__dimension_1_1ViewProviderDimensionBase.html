Detailed Description

A View Provider for the Draft Dimension object
This class is not used directly, but inherited by all dimension
view providers.

DIMENSION VIEW PROVIDER NOMENCLATURE:

    |              txt               |       e
----o--------------------------------o-----
    |                                |
    |                                |       d  
    |                                |

 a  b               c                b  a

a = DimOvershoot (vobj)
b = Arrows (vobj)
c = Dimline (obj)
d = ExtLines (vobj)
e = ExtOvershoot (vobj)
txt = label (vobj)
