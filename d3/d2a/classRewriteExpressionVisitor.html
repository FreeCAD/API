Detailed Description

The RewriteExpressionVisitor class.

A class that visits each node of an expressions, and possibly rewrites variables. This is a helper class to rewrite expressions when rows or columns are either inserted or removed, to make sure that formulas referencing cells being moved to a new locations will still be valid, i.e rewritten.
